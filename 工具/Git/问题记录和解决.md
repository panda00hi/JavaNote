# Git解决每次提交都输入账号密码
使用命令
`
git config --global credential.helper store
`
后边执行git pull/push只需再输入一次就好了。

> 注意，如果是有多个仓库对应不同账号，可以不要--global参数。

# Git强制远程覆盖本地
依次执行
``` code
    git fetch --all
    git reset --hard origin/master
    git pull
```
也可单行执行：
```
git fetch --all && git reset --hard origin/master && git pull
```
